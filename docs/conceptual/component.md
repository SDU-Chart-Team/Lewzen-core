# 组件

## 组件 Component

由图形和文本组合形成的可修改的对象，使用画布坐标系。是本核心中最顶级的维护对象。

组件由四个部分组成：描述、布局、图形矢量图、若干文本框。

- 描述：组件的一些描述性内容。包含制作时间、作者等。
- 布局：表示组件图如何呈现在画布上。包含位置、旋转。
- 图形矢量图：动态结构，表示组建中的矢量图形。
- 文本框：一个抽象结构，表示文本区域和其中内容，但并不自动维护。

### 描述

- 作者
- 制作时间
- 创建时间
- 注释
- 缩略图路径
